computing scientist and software programmer

I'm Guilherme A. Dantas, a brazilian software programmer. I have a CS degree from UFPE (Federal University of Pernambuco, Brazil, 2009), but have been programming since my teenage years. I’m a well-grounded generalist with an emphasis on backend development and functional programming.

Currently, I’m doing a lot of web development with Java (Spring, VRaptor) and tools development with Scala and Akka as a full-time Software Engineer at VivaReal.com.br. I also spend some time as a functional programming advocate, teaching Scala through example and tech talks, and speaking about the programming languages landscape in the JVM when I have the chance.

In my spare time, I like to play around with some new or less known languages (such as Kotlin, Clojure or Racket) and to challenge myself with distributed systems (recently, stream processing). I’m also interested in reactive, non-blocking systems and agile methodologies.

Despite my current position, I'm always looking for opportunities to challenge myself and learn. If you’re working on something exciting and need some help in functional programming, distributed systems or agile software development, drop me a line by email (gamsd at this domain) or ping me in any major social network.
